David Foulke

Full-Stack Engineer based in Arlington, VA

David is an experienced full-stack software engineer currently based in the Washington, D.C. metro area. He has a Bachelor's degree in Computer Science from Hampden-Sydney College in Virginia. He has several years of experience working in an agile environment, has a strong understand of data structures and algorithms, and has experience working at all stages of the development lifecycle.

Highlights and Expertise

David's first exposure to React came in the spring of 2018, when he was building a workforce planning and data visualization tool for a Federal client. The front end quickly outgrew its original architecture of HTML and jQuery, and David took it upon himself to find a more robust solution. React was best-suited for this challenge, as its lightweight footprint allowed for a more gradual refactor process. David rewrote the HTML templates as JSX, and designed a Webpack configuration that allowed the back end to serve each page as a bundled React app, removing the need for a routing module and subsequent architecture redesign. Since then, David has deepened his knowledge of React through side projects both at work and at home.
When David first began using Python in the fall of 2017, he was taken aback at the versatility and extensibility of the language, as well as by the breadth and resourcefulness of the developer community. Python offered David his first introduction to the world of MVC Web frameworks when he was tasked with building an app using the Django framework. David designed and wrote a flexible ETL system for a workforce planning application that leveraged Django's powerful ORM to churn through thousands of records at once and return reports specifically tailored to the user's request. Since then, he has even begun developing a self-configuring API schema to build on these concepts for quickly bootstrapping similar APIs with different data sources. He also likes to use Python as a multi-purpose scripting language for running data transformations and various other utilities.
David has been using TypeScript since the spring of 2019, when his project brought on a new manager who recommended the team rebuild their stack using Angular and NestJS. David's background of using strongly typed langugages in both academic and professional settings made the jump from vanilla JavaScript to TypeScript feel very natural. His experience using TypeScript spans across both front- and back-end development, and the language is now David's language of choice for new Web-based projects.
David has been a part of several projects that utilize Microsoft Azure for various aspects of the application lifecycle. Most notably, David has integrated a number of enterprise applications with the organization's Azure Active Directory system for single sign-on, and is seen as an authority on authentication solutions at his current job. David has also helped create a CI/CD workflow using Docker and Azure DevOps to deploy to an Azure cloud service.
